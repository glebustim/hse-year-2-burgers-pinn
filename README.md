В этом репозитории лежит мой ipynb ноутбук, в котором я запускал обучение моделей для написания своей курсовой работы по теме "Обучение физически информированных нейронных сетей". В ноутбуке есть фрагменты кода для смены глобальной точности в PyTorch; фуцнкция для генерации данных для обучения модели на решение уравнения Бюргерса; функции для вычисления ошибок; архитектура PINN в виде простого MLP; код для обучения моделей в разных форматах точности и рисования ландшафтов ошибок по двум весам.

В файле Pinns.txt лежат все результаты всех указанных в курсовой работе экспериментов (те же данные, которые есть в таблицах в самой работе).
